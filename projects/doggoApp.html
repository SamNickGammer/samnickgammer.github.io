Doggo Notes - A simple Notes Taking App

  • Tech Stack: Java, Android, FireBase
  • Github URL: Project Link

This is the fully customized Notes taking APP for Student, Teacher, Workers and Everyone...

In this notes App, users can access their pieces of information or the data using Firebase as well as Local Storage. This application consists of various features like:

  • Create User Account
  • Update User Account
  • Login User
  • Add, Update and Delete TODOs
  • Set Reminder on TODOs
  • Set Delete Time on TODOs
  • Add, Update and Delete Notes
  • Add all the data Directly to Cloud (FireBase)
